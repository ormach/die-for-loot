How to play

Game elements

  • Gemdice - Used as a currency to pay for items. drag and drop it on item to buy it.
  • Pile
  • Table
  • Items
  • Void
  • Bag

How to win

But 30 or more items.

Game flow

  • Each turn 4 items will be placed on board and you will get 2 gemdice.
  • Pay dice for items, use item effect to modify stuff.
  • When you run out of things to do, start the next turn.
  • Game end when you run out of items in pile.
