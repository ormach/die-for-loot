How to play

  • Each turn 4 items will be placed on table, and you will get 2 dice.
  • Dra and drop dice on items with matching cost to buy them.
  • Click on items to use their effects to modify stuff.
  • When you run out of things to do, start the next turn.

  • Game ends when you run out of items in pile.
  • Win by having 24 items in the bag at the end of the game.
  • Items — Game elements that you buy with dice. Can have passive or active effects.
  • Dice — Used as a currency to pay for items. drag and drop it on an item to buy it.

  • Pile — Contains all items at the start of the game.
  • Table — Has 6 item slots, contains items available for purchase. If table has no space, item that would be moved to table is moved to Void instead.
  • Void — Area where items go if you fail to buy things fast enough.
  • Bag — Area where you store purchased items.
